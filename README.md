# Diamond Price Prediction

## Overview

In this project, I built a machine learning model to predict the price of diamonds using features like carat, cut, color, clarity, and dimensions.

The goal of this project was to understand how an end-to-end ML pipeline works, from data preprocessing to model building and deployment.

---

## What I did

* Cleaned and prepared the dataset
* Performed feature engineering
* Trained different regression models
* Compared model performance
* Selected the best model
* Built a simple Flask app for prediction

---

## Technologies Used

* Python
* Pandas, NumPy
* Scikit-learn
* CatBoost
* Flask

---

## Project Structure

```
artifacts/      -> saved model files  
notebook/       -> experiments and analysis  
src/            -> training and pipeline code  
templates/      -> HTML files for UI  
app.py          -> main Flask app  
requirements.txt  
```

---

## How to Run

1. Clone the repository
   git clone https://github.com/kjreddy305-sys/diamond-price-prediction.git

2. Install dependencies
   pip install -r requirements.txt

3. Run the app
   python app.py

4. Open in browser
   http://127.0.0.1:5000

---

## Result

The model is able to predict diamond prices with reasonable accuracy.
Performance improved after trying different models and tuning parameters.
